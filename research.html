Large-Scale Deep Learning for Intelligent Networks
This project is supported by DOE Early Career Award [Home page].
Research Intersects AI and Network Engineering. Enabling self-driving intelligent networks that allow improved response, utilization, and reliability for exascale scientific workflows. The research builds robust networks through machine learning-based approaches, cloud computing, and software-defined networks (SDN).

Research Questions: Q1) Reduce traffic hotspots Q2) Improve network performance for workflows/applications Q3) Self-Learning Network controllers (more transfers, less congestion, etc)
